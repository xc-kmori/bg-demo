## 🚀 使用方法

### 1. 環境セットアップ
```bash
source task_manager_env/bin/activate
cd task_manager_api
```

### 2. サーバー起動
```bash
python run.py
```

### 3. 動作確認
```bash
python test_api.py
```

### 4. テスト実行
```bash
python -m pytest tests/ -v
```
